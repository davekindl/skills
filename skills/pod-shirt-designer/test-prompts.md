# POD Shirt Designer -- Phase 1 Test Prompts

**How to use:** Run each prompt through the kie.ai pipeline. Evaluate: print-readiness, text accuracy, visual appeal, and niche authenticity. Mark Pass/Fail/Partial.

**Estimated total test time:** 30 minutes (10 prompts x ~3 min each)

---

## Test 1: Typography Hybrid (P4 + S1)

**What we're testing:** The #1 selling format -- bold text + small illustration in badge

**Prompt:**
```
Vintage badge t-shirt design with arched text reading "EARN YOUR REST"
above a mountain summit silhouette with a small flag on top,
distressed texture, 2-color palette (cream and forest green),
bold sans-serif font, centered composition,
no background, transparent, isolated design element, clean edges, print-ready
```

**Expected:** Readable text from 3 feet, clean badge composition, distressed but not messy
**Fail signature:** Text garbled, layout unbalanced, background not cleanly removable

---

## Test 2: Anime Glitchcore (S4 / N1-B)

**What we're testing:** Dark anime streetwear aesthetic for Gen Z

**Prompt:**
```
Glitchcore anime streetwear t-shirt design,
hooded cyber-samurai figure with glowing blue visor,
pixel tears, digital distortion, CRT scanlines overlaying the figure,
neon pink and electric cyan, katakana text reading "DIGITAL RONIN",
dark design for black shirt,
no background, transparent, isolated design element, print-ready
```

**Expected:** Authentic anime aesthetic, readable Japanese text, glitch effects that don't overwhelm
