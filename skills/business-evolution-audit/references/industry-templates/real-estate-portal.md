# Industry Template: Real Estate Portal

Pre-built gap library for property listing platforms. Benchmark against: Zillow, Rightmove, Idealista, Hemnet, Redfin.

## Expected Features by Tier

### Table Stakes (every serious portal has these)
- Property search with location, price, size, room count filters
- Map-based search with boundary drawing
- Photo galleries (10+ per listing)
- Agent/owner contact forms
- Saved searches with email alerts
- Mobile-responsive design
- Listing categories (buy, rent, commercial)

### Tier 2 Features (market leaders have 70%+ of these)
- Price history charts per property
- Neighborhood analytics (schools, transport, crime, amenities)
- Mortgage calculator integrated in listings
- 3D virtual tours / video walkthroughs
- Street view integration
- Agent ratings and reviews
- Market trend dashboards (avg price/sqm by area, YoY change)
- Property valuation estimates (Zestimate equivalent)
- Comparable properties ("similar listings nearby")
- Recently sold data
- Floor plan drawings
- Energy efficiency ratings
- Print-quality PDF brochures per listing
- Multi-language support

### Tier 3 Features (innovators, differentiation opportunities)
- AI-powered recommendations ("homes you might like")
- AI chatbot for property search
- Virtual staging (AI-furnished empty rooms)
- Predictive pricing (future value estimates)
- Investment ROI calculator
- Rental yield estimates
- Neighborhood noise/air quality data
- School catchment area maps
- Commute time calculator (integrated with transit)
- AR room measurement
- Smart home feature tags
- Sustainability/green building filters
- Co-buying/shared ownership tools
- Legal process guide (country-specific)
- Mortgage pre-approval integration
- Moving service marketplace
- Home insurance comparison
- Automated listing quality score
- Developer project tracking (new builds)
- API for third-party integrations

## Hungarian Market Specifics

- Primary competitor set: ingatlan.com (market leader), jofogas.hu (classifieds), otthonterkep.hu (map-based), ingatlanbazar.hu (secondary)
- Regulatory: Hungarian property transactions require notary (kozjegyzo), energy certificate (energetikai tanusitvany) mandatory for sales
- Market size: ~100K active listings, ~50K transactions/year
- Price data sources: KSH (Central Statistics Office), MNB (National Bank), ingatlan.com own data
- Language: Hungarian primary, English/German secondary for Budapest luxury and Balaton vacation properties
- Key user pain point (consistent across reviews): search filter quality, listing freshness, duplicate listings, agent spam
